## Словарь синонимов
Консольное приложение для работы со словарем синонимов. Позволяет искать синонимы, добавлять новые пары слов и просматривать весь лексикон.

## Состав команды
- Гойкина Ирина (модуль A)
- Бахман Анжелика (модули B и C)

## Функциональные возможности

1. **Поиск синонима** - поиск по введенному слову с предложением добавить отсутствующее слово
2. **Добавление новой пары** - расширение словаря новыми словами и их синонимами
3. **Просмотр всех слов** - вывод всего словаря с сортировкой по алфавиту
4. **Выход из программы** - завершение работы

## Инструкция по запуску
1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать окружение и установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python master.py`
